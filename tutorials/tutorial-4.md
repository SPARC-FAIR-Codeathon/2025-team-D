# Tutorial 4: Creating plugins using generative AI from the Cloud-based development environment

## Summary
 
This tutorial guides you through creating plugins using generative AI from the Cloud-based development environment.

---

## Learning Outcomes

- Use generative AI tools to empower non-technical users to rapidly prototype a new plugins
- Test new plugins directly within the SPARC portal.

## 1. Creating a prompt for generative AI tool
1. Go to the [SPARC portal plugins page](http://130.216.217.115:3000/register-plugins).
<img width="1263" height="892" alt="Screenshot from 2025-08-05 12-47-17" src="https://github.com/user-attachments/assets/ccba3e28-e083-4e5e-9e50-afc4fb774226" />
2. Select **Generate Plugin** to enter the prompt of your plugin.
<img width="1259" height="915" alt="Screenshot from 2025-08-05 13-57-39" src="https://github.com/user-attachments/assets/f21d9324-306c-433b-bbb2-44026953a791" />
3.  Select **Generate** to Generate the plugin.
After click the **Generate** button, a hint will show up in the top of the website. It will take around 3 minutes. Enjoy your coffee during the time.

<img width="1259" height="915" alt="Screenshot from 2025-08-05 14-17-59" src="https://github.com/user-attachments/assets/40f02c86-1f3e-4e28-ad0c-fd66c2172144" />


## 2. Building the plugin
1. The new plugin will show up in the plugin list. Select **Build** to get the plugin ready.
<img width="1259" height="915" alt="Screenshot from 2025-08-05 14-26-35" src="https://github.com/user-attachments/assets/87370f36-b67b-4e85-b39f-2812e9cdb1b6" />
2. A hint will show up in the top of the website after it submitted successfully.
<img width="1259" height="915" alt="Screenshot from 2025-08-05 14-26-44" src="https://github.com/user-attachments/assets/e9ba0353-cf55-4ec8-bd7c-edd6904f0403" />
3. If the **Building** state lasts too long, click **refresh** button to refresh the state.

---
## 3. Testing the generated plugin
1. Select **Launch** to test the generated plugin.
<img width="1259" height="915" alt="Screenshot from 2025-08-05 14-29-13" src="https://github.com/user-attachments/assets/63f789ce-8236-4f63-94b2-344961d68c84" />
2. Test its functionality.
Congratulations!
You have successfully create a generated plugin!
